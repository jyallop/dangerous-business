This is a piece of algorithmic art using a data source. The source of data is in gpx format and each gps coordinate marks out my path from hiking the PCT in 2021.
Each gpx file represents one tracking session (roughly one day of hiking) and is stamped with a time, location, and elevation. The program reads all of these in, sorts them 
by date and then goes through each one sequentially essentially tracing out my path (with some skips and jumps from times I forgot to stop tracking or for getting around 
burn areas).

The title comes from the following quote from the Lord of the Rings:
“It's a dangerous business, Frodo, going out your door. You step onto the road, and if you don't keep your feet, there's no knowing where you might be swept off to.”
